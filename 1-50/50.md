# **50. Каково определение программной системы согласно OMG Essence? Каковы её части?**

Программной системы - система, состоящая из программного обеспечения, оборудования и данных, которая обеспечивает свою основную ценность посредством выполнения программ. (ну или просто в двух словах совокупность software,hardware and data)

В итоге в любой программной системе (software intensive system) есть:

- software - ПО
- hardware - аппаратное обеспечение
- data - какие то необходимые для работы данные, являющиеся частью системы.
