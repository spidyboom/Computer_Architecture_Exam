# **23. Опишите классическую схему включения электродвигателя с аварийной остановкой. Объясните, как работает обратная связь.**

Типовая схема управления электродвигателем включает в себя несколько ключевых элементов, обеспечивающих безопасную и надежную работу оборудования. Основными компонентами системы являются:

- Нормально разомкнутая кнопка Start.
- Нормально замкнутая кнопка Stop.
- Самоподдерживающееся реле.
- Аварийный выключатель EMG Stop с нормально замкнутыми контактами.

> L1 L2
> o o
> | |
> +--[\]---[\]----+--[ ]---+---------( )----+
> | ES Stop | Start | Run |
> | | | |
> | +--[ ]---+ |
> | Run |
> | |
> | |
> | |
> +------------------[ ]-------------( )----+
> | Run Motor |
> | |

Принцип работы схемы основан на формировании цепи управления через последовательное соединение защитных и управляющих элементов. При нажатии кнопки Start ток протекает через кнопку Stop, активируя катушку, которая замыкает свои контакты. Это позволяет реле оставаться включённым даже после отпускания кнопки Start, что обеспечивает работу двигателя.

Для остановки системы достаточно нажать кнопку Stop, что приводит к разрыву цепи и отключению катушки, в результате чего двигатель прекращает работу. В случае аварийной ситуации кнопка EMG Stop физически разрывает цепь, обеспечивая отключение системы даже в случае, если реле зафиксировано в активном состоянии.

Среди преимуществ данной схемы можно выделить защиту от случайного включения и гарантированное отключение при аварии.

> o o
> | |
> +--[\]---[\]----+--[ ]---+---------( )----+
> | ES Stop | Start | Run |
> | | | : |
> | +--[ ]---+ : |
> | Run : |
> | ^ : |
> | +.............../ |
> | v |
> +------------------[ ]-------------( )----+
> | Run Motor |
> | |
