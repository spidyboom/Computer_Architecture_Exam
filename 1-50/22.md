# **22. Как реализуется булев базис на электрических реле?**

Вопрос про то, как реализовать and / or / not.

Релейные диаграммы:

- Нормально разомкнутые (NO, Normally Open), обозначаемые как -[ ]-. В исходном состоянии (без подачи тока на катушку) их контакты разомкнуты, и ток через них не проходит. При активации реле контакты замыкаются, обеспечивая прохождение сигнала.
- Нормально замкнутые (NC, Normally Closed), обозначаемые как -[\]-. В исходном состоянии их контакты замкнуты, пропуская ток, а при активации реле --- размыкаются, прерывая цепь.

Актуатор - устройство на которое подается питание, чтобы оно заработало:

- -( )- (нормально неактивные)
- -(\)- (нормально активные)

AND:

```
  L1                                      L2
     o                                       o
     |             Logical AND               |
     |                                       |
     +--[ ]--------[ ]-----------( )---------+
     |  Key 1      Key 2         Door motor  |
     |                                       |
```

Две катушки реле, при подаче тока на обе, пропускают ток по проводу

OR:

>     L1                                      L2
>     o                                       o
>     |             Logical OR                |
>     |                                       |
>     +----+--[ ]--------------+--( )---------+
>     |    |  Exterior unlock  |  Unlock      |
>     |    |                   |              |
>     |    +--[ ]--------------+              |
>     |        Interior unlock                |
>     |                                       |

Две катушки паралельно, подали на одну из них (ну или на две сразу) - ток пойдет по проводу

NOT:

>     L1                                      L2
>     o                                       o
>     |             Logical NOT               |
>     |                                       |
>     +--[ ]----------------------(\)---------+
>     |  Key 1                    Door motor  |
>     |                                       |

При активации реле контакт размыкается, прерывая цепь. Таким образом, индикатор горит, пока дверь закрыта, и гаснет при её открытии, обеспечивая инверсию сигнала.
