# **124. Какие механизмы необходимы для реализации кооперативной многозадачности?**

Механизмы:

- остановки: задача добровольно передает управление диспетчеру
- сохранения состояния задачи: регистры, окружение
- возобновление: восстановить состояние, на котором задача прервалась, передать ей управление
- планирования: диспетчеризация порядка выполнения задач

Дополнительные механизмы:

- изоляция задач: задачи выполняются независимо относительно друг друга
- взаимодействие между задачами: ну тут понятно
