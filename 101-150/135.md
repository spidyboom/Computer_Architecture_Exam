# **135. Какие виды событий существуют в системе прерываний (по фронту, по уровню, по сообщению, doorbell)? В чём особенности каждого?**

Виды прерываний:

- По фронту - изменение уровня от 0 к 1 и наоборот
- По уровню - Пока на линии некоторый уровень фиксируем прерывание
- По сообщению - Нету паровода напрямую, устройство генерирует сообщение записи в память (напрример шины PCIe)
- По дверному звонку (doorbell) - Внутри контроллера прерываний выделяется специальный аппаратный регистр, жестко закрепленный за конкретным ядром или очередью задач - Doorbell Register. Внешнее устройство выполняет быструю операцию записи в этот регистр. Факт записи любого значения в эту ячейку физически звонит в звонок - мгновенно активирует триггер аппаратного прерывания для целевого ядра.
