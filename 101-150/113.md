# **113. Каковы шаги типового взаимодействия с устройством ввода-вывода? Каков интерфейс устройства для процессора?**

1. Конфигурирование устройства доступа (Объясняем внешнему устройству что мы будем с ним делать)
2. Проверка состояния (Готово ли оно?)
3. Отправка данных
4. Проверка статуса ответа (Можем ли читать ответ?)
5. Чтение ответа

Интерфейс ввода-вывода состоит из:

- Регистров данных (обмен данными с ву)
- Регистров статуса (Готово/Нет)
- Протоколов взаимодействия
