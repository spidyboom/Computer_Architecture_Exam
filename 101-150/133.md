# **133. Как классифицируются прерывания по источнику (синхронные/асинхронные, аппаратные/программные)? Их отличительные особенности.**

`Аппаратные` (железо):

- `Внешние` (`асинхронны` относительно инструкций процессора) - пакет по сети, нажатие клавиши
- `Внутренние` (`синхронны` с тактами процессора) - деление на ноль, any ошибка

`Программные` (вызваны ПО) - вызов диспетчера через программное прерывание
