# **150. Что такое контроллер прямого доступа к памяти (DMA)? Каковы его назначение, интерфейс, достоинства и недостатки?**

ЧИТЕРЫ СУКИ ЕБАНЫЕ ГОРЕТЬ В АДУ ПИДАРАСАМ

ПДПстольгия

`DMA` (`Direct Memory Access`) - аппаратный модуль, позволяющий устройствам ввода-вывода обмениваться данными с оперативной памятью минуя CPU. Является заменой проца ввода-вывода. DMA просто копирует данные из одной области в другую

Достоинства:

- DMA может быть заточен под память - по частоте, системе команд, организации работы с памятью и устройствами.
- Интерактивность, так как процессор разгружен от "рутины"
- Параллелизм: DMA может иметь несколько каналов для параллельной работы

Недостатки:

- Проблемы совместимости
- Сложность при непоследовательном доступе к памяти
- Ограниченный контроль за системной шиной, синхронизация работы процессора и DMA
- Конфликт использования DMA разными устройствами ввода-вывода
