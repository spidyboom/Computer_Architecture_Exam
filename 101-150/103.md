# **103. Как разрешаются структурные конфликты в конвейере?**

Способы:

- Гарвардская архитектура (чтение инструкции и чтение операнда не конфликтуют по причине их разделения)
- Двухпортовая память (несколько каналов доступа к памяти)
- Пузырек. Простой между этапами, который разносит конфликтующие этапы по времени.
