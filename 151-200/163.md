# **163. Что такое кеш? Каково его назначение, место в иерархии памяти и основные метрики?**

`Кеш` - буфер с быстрым доступом, содержащий объем памяти, которая может быть зарпошена повторно с наибольшей вероятностью.

Допустим время доступа к памяти со стороны процессора будет 100ms, в кэш 20ms, а из кэша в память 90ms. Как видим нам круто считать повторную инфу из кэша, нежели просто из памяти.

Место в иерархии - пярмо за регистрами процессора

Метрики:

1. Объем кэша
2. Скорость доступа
3. Размеры кэш-линий
4. Частота cashe hit-ов
5. Уровень ассоциативности
