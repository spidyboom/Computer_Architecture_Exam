# **171. Что такое множественно-ассоциативный кеш и что определяет уровень множественности?**

`Множество-ассоциативный кеш` (`Set-Associative Cache`) представляет собой компромисс между полностью ассоциативным и прямым отображением. Он сочетает преимущества обоих подходов:

- Внутри - это несколько кешей с прямым отображением, работающих параллельно
- Снаружи - действует как ограниченно ассоциативный кеш: каждый блок памяти может быть размещён в одном из нескольких вариантов.

Например, при 2-канальной ассоциативности блок может попасть в один из двух слотов внутри набора, а при 8-канальной - в один из восьми.
