# **61. Что такое универсальный процессор? Каковы его особенности и свойства?**

`Информационный процессор` - это обобщённое название процессора, предназначенного для обработки информации.

`Универсальный процессор` - это вычислительное устройство, построенное по принципу двухэтапного производства и способное адаптироваться к решению конкретных практических задач.

Для уточнения понятия универсальности необходимо учитывать несколько важных аспектов:

1. Универсальность ≠ полнота процессора по Тьюрингу.

- Универсальность определяется широтой круга решаемых задач, а не вычислительной полнотой.

2. Теоретическая универсальность ≠ практическая универсальность.

- Теоретически универсальные системы (например, Машина Тьюринга, язык Brainfuck) могут выполнять любые вычисления, но их практическая применимость ограничена.

3. Универсальность противоречит эффективности.

- Чем универсальнее процессор, тем ниже его эффективность, и наоборот.

4. Современный процессор.

- Почти всегда совокупность программного и аппаратного обеспечения.

Характерные свойства универсального процессора:

1. Двухэтапное производство.

- Жизненный цикл процессора разделён на этапы производства (создание аппаратной составляющей) и эксплуатации (использование процессора для решения практических задач).

2. Полнота по Тьюрингу.

- Свойство не является обязательным требованием, но в большинстве случаев процессоры обладают полнотой по Тьюрингу.

3. Отсутствие серьёзных ограничений на объём программы.

- Объём памяти, выделенный для команд, напрямую ограничивает сложность исполняемой программы. Например, при объёме памяти команд в один килобайт размещение сложной программы технически невозможно.

4. Изменяемость программного обеспечения после его реализации.

- Изменяемость программной части позволяет адаптировать процессор под новые задачи в процессе эксплуатации.
