# **66. Что такое машина фон Неймана? Какова её связь с машиной Тьюринга? Каковы её ключевые принципы?**

`Машина фон Неймана` - базовая архитектура компьютера (принцип совместного хранения команд и данных в памяти компьютера):

1. В основе машины фон Неймана лежит Central Process Unit (CPU) - универсальный процессор, способный решать любые задачи благодаря максимальной гибкости и программируемости. Он включает в себя:

- Control Unit - блок управления, координирующий выполнение инструкций.
- Arithmetic/Logic Unit (ALU) - арифметико-логическое устройство, обрабатывающее данные.

2. Этот CPU обменивается данными с Memory Unit, представляющим из себя Random Access Memory (главное отличие от Тьюринга, ведь там была лента)
3. Все вышеперечисленное объединено в одну структуру, слева и справа от которой есть Input Device и Output Device соответственно

Основные принципы:

- Инструкции и данные объединены
- Призвана быть максимально простой в реализации и производстве
- Последовательное выполнение
- Использование двоичного кодирования
- Ячейки памяти имеют адреса
- Возможность условного перехода
