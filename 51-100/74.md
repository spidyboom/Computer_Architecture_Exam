# **74. Что такое "аккумуляторная система команд"? Каковы основные элементы? Приведите примеры. Каковы достоинства и недостатки?**

`Аккумуляторная система команд` - это архитектура набора команд, в которой процессор имеет ровно один специальный регистр общего назначения, называемый аккумулятором. Он по умолчанию и неявно является как одним из источников данных (операндов) для АЛУ, так и местом, куда автоматически записывается результат любой операции. Основные элементы: аккумулятор, алу, память

Достоинства:

- мало опкодов, мало занимает сама инструкция и фикс размер
- аппаратурная реализация тривиальная
- легкость программирования (тк как и правда тривиально)

Недостатки:

- Низкая производительность
- Сложность параллелизма и конвейеризации
- Крайне много обращений к памяти

Примеры:

- Всякое старье типа IBM 7090, DEC PDP-8
- DSP(digital signal processor) архитектура
- бэвмачка наша любимая сладкая вкусная

```
A = B + (C * D)

load C
mul D
add B
store A
```
