# **80. В каких случаях CISC инструкции эффективны? Приведите примеры.**

CISC процессора выигрывают там, где одна сложная команда заменяет несколько простых будь то операции со строками, математические вычисления, системные вызовы

Вот эта команда съест 5 тактов (вроде)

```
add #01 <- 34 + #03      5t
```

Вот эти съедят 8 тактов на троих (вроде)

```
acc <- 34                2t
acc <-+ #03              3t
#01 <- acc               3t
```
