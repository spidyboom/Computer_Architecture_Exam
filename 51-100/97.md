# **97. Как сравниваются CISC и RISC с точки зрения архитектуры?**

CISC:

- register to memory
- memory to memory

RISC:

- register to register
- больше регистров

CISC:

- Переменная длина инструкций и большое их количество

RISC:

- Фиксированная длина инструкций и простота

CISC:

- увеличенные возможности работы с памятью read-modify-write

RISC:

- онли LOAD | STORE
